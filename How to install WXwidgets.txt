Download zip from https://www.wxwidgets.org/downloads/
unpack zip where you want
go to %WXWIN%\build\msw and open wx_vc12.sln
Build solution
To test your build go to %WXWIN%\samples\minimal and open a .vxproj file
Build solution and run program, when errors wxWidgets isn't build right

Add a environment variable named Wxwidgets and the value is the path to your to folder of wxWidgets
(go to system and then advanced settings to add a environment variable)

Now you can build TTT GUI
